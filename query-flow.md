# Query Flow: Frontend to Backend

```
┌─────────────────────────────────────────────────────────────────────┐
│                            FRONTEND                                  │
│                                                                      │
│   User types message  ──►  sendMessage()  ──►  POST /api/query      │
└─────────────────────────────────────┬───────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────┐
│                             FASTAPI                                  │
│                                                                      │
│   Create session if needed  ──►  rag_system.query()                 │
└─────────────────────────────────────┬───────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────┐
│                           RAG SYSTEM                                 │
│                                                                      │
│   1. Get conversation history  (SessionManager)                      │
│   2. Call AIGenerator with query + history + tools                   │
└──────────────────────┬──────────────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────────────┐
│                        CLAUDE API  (1st call)                        │
│                                                                      │
│          ┌─────────────────┐         ┌─────────────────────┐        │
│          │ General question│         │  Course question     │        │
│          │                 │         │                      │        │
│          │  Answer directly│         │  Call search tool    │        │
│          └────────┬────────┘         └──────────┬──────────┘        │
└───────────────────┼──────────────────────────────┼──────────────────┘
                    │                              │
                    │              ┌───────────────▼──────────────┐
                    │              │         VECTOR STORE          │
                    │              │                               │
                    │              │  Resolve course name (fuzzy)  │
                    │              │  Semantic search in ChromaDB  │
                    │              │  Return matching chunks       │
                    │              └───────────────┬──────────────┘
                    │                              │
                    │              ┌───────────────▼──────────────┐
                    │              │      CLAUDE API  (2nd call)   │
                    │              │                               │
                    │              │  Synthesize answer from       │
                    │              │  retrieved chunks             │
                    │              └───────────────┬──────────────┘
                    │                              │
                    └──────────────┬───────────────┘
                                   │
                                   ▼
┌─────────────────────────────────────────────────────────────────────┐
│                           RAG SYSTEM                                 │
│                                                                      │
│   3. Save exchange to session history  (SessionManager)              │
│   4. Return answer + sources                                         │
└─────────────────────────────────────┬───────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────┐
│                            FRONTEND                                  │
│                                                                      │
│   Render answer as Markdown  ──►  Show sources  ──►  User sees reply │
└─────────────────────────────────────────────────────────────────────┘
```
